# ⚔️ Феникс RPG — Discord Bot

Текстовая RPG прямо в Discord. Сражения, прокачка, PvP дуэли, гильдии, казино, карточная система и многое другое.

---

## Возможности

- **Бои с монстрами** — 9 локаций, 25+ уникальных монстров с особыми способностями
- **Прокачка персонажа** — уровни, классы, расы, профессии, питомцы
- **PvP дуэли** — вызов другого игрока на бой
- **Квестовая система** — цепочки квестов с наградами
- **Гильдии** — создавай и вступай в гильдии, пополняй казну
- **Банковская карта** — 7 тиров (Бронзовая → Архаловская) с бонусами
- **Рынок** — торговля предметами между игроками
- **Казино** — кости и блэкджек
- **Исследование локаций** — случайные события, клады, ловушки, торговцы
- **Достижения** — 20+ ачивок
- **Улучшение снаряжения** — прокачка оружия и брони
- **Красивые карточки** — все ответы бота с графикой на @napi-rs/canvas
- **Главное меню `/menu`** — все действия в одном месте с визуальной карточкой персонажа
- **Ежедневные снапшоты** — автоматическое резервное копирование статов игроков

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать приключение (выбрать класс) |
| `/profile` | Карточка профиля |
| `/fight` | Сразиться с монстром |
| `/shop` | Магазин предметов и карт |
| `/inventory` | Инвентарь |
| `/use` | Использовать предмет |
| `/daily` | Ежедневная награда |
| `/quest` | Текущий квест |
| `/explore` | Исследовать локацию |
| `/location` | Сменить локацию |
| `/duel` | Вызвать игрока на дуэль |
| `/gamble` | Казино (кости / блэкджек) |
| `/top` | Топ игроков |
| `/weekly` | Недельный рейтинг |
| `/achievements` | Достижения |
| `/stats` | Детальная статистика |
| `/customize` | Сменить расу |
| `/profession` | Выбрать профессию |
| `/pet` | Купить питомца |
| `/guild` | Управление гильдией |
| `/card` | Банковская карта |
| `/deposit` / `/withdraw` | Пополнить / снять с карты |
| `/market` | Рынок игроков |
| `/trade` | Передать карту игроку |
| `/upgrade` | Улучшить снаряжение |
| `/rest` | Восстановить HP |
| `/menu` | Главное меню — все действия в одном окне |
| `/help` | Список команд |

## Структура проекта

```
rpg-bot/
├── index.js                    # точка входа, Discord client
├── src/
│   ├── data/constants.js       # игровые константы
│   ├── db/
│   │   ├── database.js         # инициализация SQLite
│   │   └── queries.js          # prepared statements, хелперы
│   ├── state.js                # общее состояние (бои, кэши)
│   ├── canvas/
│   │   ├── helpers.js          # утилиты рисования
│   │   └── generators.js       # генерация карточек
│   └── handlers/
│       ├── commands.js         # slash-команды
│       ├── buttons.js          # кнопки
│       ├── select.js           # выпадающие меню
│       └── autocomplete.js     # автодополнение
```

## Установка

```bash
git clone https://github.com/DimaTor5/rpg-bot.git
cd rpg-bot
npm install
cp .env.example .env
# заполни .env своими данными
node index.js
```

## Переменные окружения

```env
DISCORD_TOKEN=  # токен бота из Discord Developer Portal
CLIENT_ID=      # ID приложения
DB_PATH=        # путь к SQLite файлу (по умолчанию /var/www/rpg-bot/rpg.db)
```
